# DKCRandomKey
Random key generator for Donkeycar S4 discount 

## How to use

```bash 
git clone https://github.com/yoyojacky/DKCRandomKey.git
cd DKCRandomKey/
sudo apt update && sudo apt -y install virtualenv 
virtualenv -p python3 venv 
cd venv 
source bin/activate 
pip install -r ../requirement.txt 
python donkey_gen_key.py 
```
